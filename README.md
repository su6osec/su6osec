## About Me

Cybersecurity researcher & bug bounty hunter focused on web application security, reconnaissance, and developing automation tools for penetration testing. I believe in open-source security tooling that empowers both newcomers and experienced practitioners.

## su6osec Toolkit

Professional cybersecurity tools I’ve built for the community. Every tool features production-quality code, CLI + web interfaces, comprehensive documentation, and Docker support.

| Tool | Description | Type |
|------|-------------|------|
| [subfinder-pro](https://github.com/su6osec/subfinder-pro) | Advanced subdomain enumeration (CT, API, brute-force) | Recon |
| [portsentry](https://github.com/su6osec/portsentry) | Async port scanner with service detection & CVE correlation | Scanning |
| [webvulnscan](https://github.com/su6osec/webvulnscan) | Web app vulnerability scanner (XSS, SQLi, redirects) | Scanning |
| [api-security-toolkit](https://github.com/su6osec/api-security-toolkit) | JWT analysis, IDOR detection, rate limiting, auth bypass | Testing |
| [payloadforge](https://github.com/su6osec/payloadforge) | Payload generator & encoder for XSS, SQLi, RCE, SSRF | Offensive |
| [recon-hq](https://github.com/su6osec/recon-hq) | Centralized recon dashboard with SQLite + FastAPI | Dashboard |
| [passaudit](https://github.com/su6osec/passaudit) | Password strength analyzer & breach database checker | Audit |
| [ssl-scout](https://github.com/su6osec/ssl-scout) | SSL/TLS config analyzer with SSL Labs-style grading | Audit |
| [bounty-report](https://github.com/su6osec/bounty-report) | Professional bug bounty report generator (H1/Bugcrowd) | Reporting |
